When configuring a workstation, buying the most expensive CPU doesn't always guarantee the best performance. Different software pipelines utilize hardware in fundamentally different ways.

## Clock Speed vs. Core Count

1. **Active Modeling (CAD/BIM)**: Software like SolidWorks, Revit, and AutoCAD primarily operate on a single thread during active modeling. For these tasks, high clock speeds (5GHz+) are vastly more important than core count.
2. **Rendering & Simulation**: Engines like V-Ray, Arnold, and Blender Cycles scale near-perfectly across multiple cores. This is where AMD's Threadripper Pro processors excel.

## The Hybrid Approach

For artists who model *and* render on the same machine, a balanced approach is necessary. Systems built around 24-core or 32-core Threadrippers offer high enough boost clocks for smooth viewport performance, while retaining enough compute density for overnight rendering.

### RAM Configuration

Do not neglect your memory bandwidth. Always populate all available memory channels (8 channels on Threadripper Pro) to maximize data throughput to the CPU.
